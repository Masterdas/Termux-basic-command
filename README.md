### **File & Directory Commands**  
- `ls` тЖТ List files and directories  
- `ls -a` тЖТ Show hidden files  
- `pwd` тЖТ Show the current directory  
- `cd <directory>` тЖТ Change directory  
- `cd ..` тЖТ Move to the previous directory  
- `mkdir <folder_name>` тЖТ Create a new folder  
- `rm <file_name>` тЖТ Delete a file  
- `rm -rf <folder_name>` тЖТ Delete a folder with all files  
- `cp <source> <destination>` тЖТ Copy a file  
- `mv <source> <destination>` тЖТ Move or rename a file  

### **Package Management**  
- `pkg update` тЖТ Update package list  
- `pkg upgrade` тЖТ Upgrade all packages  
- `pkg install <package_name>` тЖТ Install a package  
- `pkg uninstall <package_name>` тЖТ Remove a package  
- `pkg list-all` тЖТ List all available packages

- `cat <file>` тЖТ Display file content  
- `nano <file>` тЖТ Edit a file using nano editor
