### **File & Directory Commands**  
- `ls` â†’ List files and directories  
- `ls -a` â†’ Show hidden files  
- `pwd` â†’ Show the current directory  
- `cd <directory>` â†’ Change directory  
- `cd ..` â†’ Move to the previous directory  
- `mkdir <folder_name>` â†’ Create a new folder  
- `rm <file_name>` â†’ Delete a file  
- `rm -rf <folder_name>` â†’ Delete a folder with all files  
- `cp <source> <destination>` â†’ Copy a file  
- `mv <source> <destination>` â†’ Move or rename a file  

### **Package Management**  
- `pkg update` â†’ Update package list  
- `pkg upgrade` â†’ Upgrade all packages  
- `pkg install <package_name>` â†’ Install a package  
- `pkg uninstall <package_name>` â†’ Remove a package  
- `pkg list-all` â†’ List all available packages  

### **Text & File Operations**  
- `cat <file>` â†’ Display file content  
- `nano <file>` â†’ Edit a file using nano editor  
- `echo "text" > file.txt` â†’ Create a file with text  
- `touch <file>` â†’ Create an empty file  

### **User & System Info**  
- `whoami` â†’ Show the current username  
- `uname -a` â†’ Show system information  
- `termux-info` â†’ Show Termux system info  

### **Networking**  
- `ping <website>` â†’ Check network connection  
- `curl <url>` â†’ Fetch webpage content  
- `wget <url>` â†’ Download a file from the internet  

### **Process & Task Management**  
- `ps` â†’ Show running processes  
- `kill <process_id>` â†’ Kill a process  
- `top` â†’ Show real-time running processes  

### **Storage & Permissions**  
- `termux-setup-storage` â†’ Grant storage access  
- `df -h` â†’ Check available storage  
- `du -sh <folder>` â†’ Check folder size
